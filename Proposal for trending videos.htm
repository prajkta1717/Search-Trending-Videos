Search trending videos

 

Trending videos help people discover new videos they might be interested in. One of the main reasons why I chose to work on this project is because there is a plethora of data available in this dataset. There are a lot of inferences we can be drawn from the views and likes on Trending Videos. This project piques my interest because there are many similar websites and I wish to learn the skills needed to build such an application.

 

Features: 

  • Display trending videos on the home page
  • Search videos based on keywords
  • Sort videos based on views and likes
  • Display information about the video

 

Display trending videos on the home page:

If the User has not entered anything in the search field, then the most trending videos and its information will be displayed.

 

Search videos based on keywords:

When you enter keywords in the search field, it matches with the dataset and provides the relevant video results.

 

Sort videos based on views and likes:

Functionality for sorting search results based on likes and views is also provided

 

Display information about the video:

Along with the video, search will also display the information about the video such as channel details, date of upload, number of views and number of likes.
